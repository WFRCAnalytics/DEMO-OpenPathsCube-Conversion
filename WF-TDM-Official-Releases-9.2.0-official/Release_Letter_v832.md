·  Model users are encouraged to contact WFRC or MAG staff prior to any application of the model to request an in-person or virtual consultation meeting to discuss planned uses, suitable methodology, technical questions, and to ensure possession of the most current model release.

·  Model users may choose to make modifications to the model at their own discretion and risk, and are strongly encouraged to preserve a record of all changes and disclaim the presence and nature of all changes made when sharing or communicating any results.

·  Model users, at their earliest convenience, are strongly encouraged to provide feedback regarding model performance directly to WFRC or MAG staff. Similarly, requests for future model enhancements are encouraged.

·  Model users are requested not to redistribute this model but instead to refer any interested organizations or individuals to make a similar request for the model directly to WFRC or MAG staff. This will facilitate use of the most current model, opportunities for appropriate technical discussions, and additions to the Model User Group email list.

·  The 8.3.2 release of the model contains multiple networks that represent the road/highway and transit system in base (2015), existing (2019), and future years (2024, 2030, 2040, & 2050). The networks reflect funded Transportation Improvement Program (TIP) projects, both 'fiscally constrained' and 'need based' projects in the adopted 2019 Regional Transportation Plans (RTPs), and RTP amendments to-date. Additionally in the WFRC area of the model space, socioeconomic projections are included for both the 'fiscally constrained' and 'needs based' transportation investment scenarios.  

Accordingly, here is the WF TDM v8.3.2 - 2022-02-04 repository on Github.  If you have any issues accessing through Github, please let me know and I'll be happy to arrange for another delivery method.

How to Access the Wasatch Front Travel Demand Model via GitHub:

If you do not already have one, create a GitHub account.
Email your GitHub email or username to me, and I will then add you to the TDM repository with read-only access.
You will receive an email from GitHub saying that you’ve been invited to WFRCAnalytics/WF-TDM-v8.3.2-2022-02-04
Click View invitation to be redirected to GitHub.
Click Accept invitation to accept access to the repository.
You now have access to the TDM repository at https://github.com/WFRCAnalytics/WF-TDM-v8.3.2-2022-02-04

Note: Within this GitHub repository, you can customize Watch notifications to receive automatic notification whenever files in the repository change. For example, if there is an updated master network or a script bug fix, you can receive an email alerting you to the change. You can then access the repository and download the new files.
